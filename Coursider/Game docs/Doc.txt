Цель игры:
Игра будет выполнена в стиле бесконечного геймплея. Цель игрока набрать максимальное количество очков рейтинга.
Очки набираются путём преодоления препятствий и уничтожения противников.

Геймплей:
Игрок всеми доступными способами перемещается вверх и уничтожает врагов на пути. За каждый пройденный метр вверх
и за каждого убитого игрока начисляются очки рейтинга, а также меняется локация. Если игрок падает вниз, он не теряет здоровья,
но теряется сумма метров пройденного пути. По пути игроку будут встречаться бонусы.

Умения игрока:
1. Прыгать (+ двойной прыжок).
2. Передвигаться (влево/вправо).
3. Притягивать к себе объекты.
4. Притягиваться к объектам.
5. Ставить щит.

Умения противников:
(Противники будут разные в зависимости от локации, на которой находится игрок)
Противник 1: Передвигаться по плоскости + ближний бой
Противник 2: Передвигаться по плоскости + дальний бой
Противник 3: Передвигаться по плоскости + дальний бой + ближний бой
Противник 4: Передвигаться по всем осям + ближний бой
Противник 5: Передвигаться по всем осям + дальний бой + ближний бой
Противник 6: Следовать за игроком + летать + дальный бой + ближний бой
